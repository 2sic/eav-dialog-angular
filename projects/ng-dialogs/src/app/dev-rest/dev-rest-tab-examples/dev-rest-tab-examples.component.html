menu_book Virtual REST URLs are short. They will be resolved internally to the full length (based on the dnn-version, current portal, etc.) thanks to $2sxc JS API. This happens when you use the $2sxc(id-or-htmlnode).webApi.get(...) or if you use dnn-sxc-angular (npm package) (which also passes url-resolving through $2sxc).
